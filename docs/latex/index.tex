\hypertarget{index_autotoc_md2}{}\doxysubsubsection{Project Overview and Description}\label{index_autotoc_md2}
Since robots are being rapidly deployed for various domestic applications, Acme Robotics is looking to develop a domestic robot as a part of its 5-\/year product road-\/map. The robot called the \char`\"{}\+Domestic Decluttering Robot\char`\"{} will declutter the house floor by identifying different items, collecting them, and placing them at their desired location in the house. The robot will scout the house floor to detect and grasp the desired object. Once grasped, it will place the object at the predefined location. The robot will have a camera to identify the object, Lidar sensor to make a 3D map of the environment around it, and a 8 D\+OF robotic manipulator mounted on it.\hypertarget{index_autotoc_md3}{}\doxysubsubsection{Documentation}\label{index_autotoc_md3}

Install Tiago packages from \href{http://wiki.ros.org/Robots/TIAGo/Tutorials/Installation/InstallUbuntuAndROS}{\texttt{ wiki.\+ros.\+org}}


\begin{DoxyCode}{0}
\DoxyCodeLine{cd <catkin workspace>/src}
\DoxyCodeLine{sudo apt-\/get install git}
\DoxyCodeLine{git clone -\/-\/recursive https://github.com/HrushikeshBudhale/decluttering\_domestic\_robot.git}
\DoxyCodeLine{cd ..}
\DoxyCodeLine{catkin build}
\DoxyCodeLine{source ./devel/setup.bash}
\end{DoxyCode}


Copy the existing S\+L\+AM configuration files and world file from cloned repo to the tiago directory using following command. 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo cp -\/r src/decluttering\_domestic\_robot/configuration/smallHouse/ \string~/.pal/tiago\_maps/configurations/}
\DoxyCodeLine{sudo cp src/decluttering\_domestic\_robot/world/smallHouse.world /tiago\_public\_ws/src/tiago\_simulation/tiago\_gazebo/worlds/}
\end{DoxyCode}


For launching the simulation demo run following command. This will start a gazebo environment of a house with 5 rooms. Tiago robot will be spawned. Also a small blue color box will be spawned in a random location. 
\begin{DoxyCode}{0}
\DoxyCodeLine{roslaunch decluttering\_domestic\_robot simulation.launch}
\end{DoxyCode}


Launch file can be used to record a bag file by passing {\ttfamily rocord\+\_\+bag} argument as follows 
\begin{DoxyCode}{0}
\DoxyCodeLine{roslaunch decluttering\_domestic\_robot simulation.launch record\_bag:=true}
\end{DoxyCode}
